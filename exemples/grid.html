Grid

Introdução às Grids do Structure

Grids Structure são fáceis de trabalhar, e muito poderosas. Existem alguns conceitos simples para manter em mente:


Linha contra classes de Colunas
Grids Structure consistem em dois tipos de classes: Classe de linha ( row ) e classes de coluna ( col-l-* ou col-s-* )


Col Large x Col Small
As Colunas Structure são dividida em dois tipos: large (col-l-*) e small (col-s-*), esses dois tipos servem para diferenciar colunas em telas largas e em telas pequenas, o que permite por exemplo alterar a largura da col em cada tamanho de tela.
Usando somente a col-l-* quando a pagina for acessada por um dispositivo de tela pequena a col ficará com sua largura em 100%, já usando a col-s- você pode definir a largura de acordo com a sua necessidade.


As larguras das colunas são frações
Colunas têm vários nomes de classe que representam suas larguras. Por exemplo, col-1-2 tem uma largura de 50%, enquanto col-1-5 teria uma largura de 20%.


Todos os elementos filho de uma linha deve ser coluna
Elementos filho contidos dentro de um elemento com classe row deve ser uma coluna ( col-l-* ou col-s-* ).


Conteúdo vai dentro de unidades de grade
Todo o conteúdo que é visível para as pessoas deve ser contido dentro de um elemento com classe row, isso garante que o conteúdo será processado corretamente.
